# softsled.github.io

The website for **SoftSled** — an open-source software Windows Media Center Extender.

This is a plain static HTML/CSS site (no build step). It's served by GitHub Pages
directly from this repository.

## Pages

| File | Purpose |
|------|---------|
| `index.html` | Home / landing page |
| `getting-started.html` | Prerequisites, host setup, pairing walkthrough |
| `features.html` | Feature overview and capability matrix |
| `how-it-works.html` | Technical architecture (pairing, DSLR, FreeRDP, RTSP A/V) |
| `downloads.html` | Links to the SoftSled2 GitHub Releases |
| `assets/css/style.css` | Shared stylesheet (Windows Media Center–inspired theme) |
| `assets/img/` | Screenshots |

## Editing

Everything is hand-written HTML — open a page in a browser to preview, edit, commit.
All asset links are relative so the site works regardless of the Pages base URL.
`.nojekyll` disables Jekyll processing so the files are served verbatim.

Download links point to the main project's releases:
<https://github.com/L2N6H5B3/SoftSled2/releases>
