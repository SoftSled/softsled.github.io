Downloads

SoftSled comes in two parts. Grab the Host Setup tool for your Media Center PC and the Extender app for the PC you'll watch from. Both are published as releases on GitHub.

🛠️

SoftSled Host Setup

Run this once on your Media Center host PC. It installs the SoftSled CA certificate and patches the host's own Mcx2Prov.exe in place so it will pair with SoftSled — and reverses both changes when you uninstall.

Windows · run as administrator · fully reversible

Prerequisite: the host PC needs .NET Framework 4.0.3019 (the .NET 4 Platform Update). Grab it from Microsoft's download page and install it before running the tool.
Download from Releases  →
📺

SoftSled Extender

Run this on the PC you want to watch from. It pairs with Media Center, then renders the full 10-foot experience — live & recorded TV, music, pictures and video.

Windows · the main SoftSled application

Which do I install where?

If this PC…Install
has Windows Media Center, your TV tuners and recordings (the host)SoftSled Host Setup
is the one you want to watch Media Center on (the extender)SoftSled Extender

New here? The Getting Started guide walks through installing both and pairing them in order.

Build from source

SoftSled is open source. Clone the repository, open the solution in Visual Studio and build the Extender and Host Setup projects yourself.
